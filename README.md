This is my undergraduate thesis, done in the year of 2022. My thesis focuses on usability analysis of NFT marketplaces. The thesis contributions are as follows :
1. Evaluation of five different NFT Marketplaces
2. Semi-structured interviews with individuals new to blockchain and NFTs
3. Semi-structured interviews with users of NFTs
4. Cognitive walkthrough with novice users of Opensea
5. Ideal design recommendation for an NFT marketplace
